FAQ

Punya Pertanyaan?

Kami sudah siapkan jawabannya. Kalau masih ada yang kurang jelas, silakan hubungi kami.

Apakah file bisa diedit sesuai kebutuhan sekolah?+
Bisa. Semua file dalam format Word (.docx) sehingga bebas diedit sesuai data sekolah, jadwal, dan peserta didik.
Untuk jenjang apa saja paket ini tersedia?+
Tersedia untuk TK/PAUD, SD, SMP, dan SMA, masing-masing disusun mengikuti karakteristik jenjangnya.
Bagaimana cara menerima file setelah membeli?+
Setelah pemesanan dikonfirmasi, seluruh file langsung dikirim dan bisa langsung diunduh.
Apakah sesuai Kurikulum Merdeka terbaru?+
Ya, seluruh dokumen mengikuti regulasi dan Kurikulum Merdeka terbaru.
Apakah bisa dipakai untuk tahun ajaran berikutnya?+
Bisa, file dapat digunakan berulang dan disesuaikan setiap tahun ajaran baru.
