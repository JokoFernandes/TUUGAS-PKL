FAQ

Pertanyaan yang Sering Ditanyakan

Masih ada yang ingin ditanyakan? Berikut jawaban seputar paket ini.

Apakah file bisa diedit sesuai kebutuhan sekolah?+
Bisa. Semua file dalam format Word (.docx) sehingga guru bebas mengedit nama sekolah, jadwal, nama peserta didik, dan bagian lain sesuai kebutuhan.
Untuk jenjang apa saja paket ini tersedia?+
Paket tersedia untuk jenjang TK/PAUD, SD, SMP, dan SMA, masing-masing disusun mengikuti karakteristik dan kurikulum jenjangnya.
Bagaimana cara menerima file setelah membeli?+
Setelah pemesanan dikonfirmasi, seluruh file akan dikirimkan langsung dan dapat langsung diunduh dan digunakan.
Apakah paket ini sesuai dengan Kurikulum Merdeka terbaru?+
Ya, seluruh dokumen disusun dan diperbarui mengikuti regulasi dan Kurikulum Merdeka terbaru.
Apakah bisa digunakan untuk tahun ajaran berikutnya?+
Bisa. File dapat digunakan berulang kali dan disesuaikan kembali setiap tahun ajaran baru.
Apakah ada bantuan jika mengalami kendala?+
Tersedia support admin yang siap membantu jika ada pertanyaan atau kendala dalam penggunaan file.
